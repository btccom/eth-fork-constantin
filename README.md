### Getting started
#### Install dependencies

```
yarn
```
or
```
npm install
```

### Usage

#### Start project on local server (go to localhost:1333)

```
yarn run dev
```
or
```
npm run dev
```

#### Build project (in "dist" folder):

```
yarn run build
```
or
```
npm run build
```

#### Preview minified production-ready bundle (go to localhost:3000)

```
yarn run preview
```
or
```
npm run preview
```
